## 문제 1. 정수 10개 입력받아 평균 구하기

## 문제 설명
사용자로부터 정수 10개를 입력받아 배열에 저장한 뒤,
입력된 정수들의 평균값을 출력하는 프로그램을 작성하시오.

## 조건
정수는 총 10개 입력받는다.
입력받은 값은 배열에 저장한다.
평균은 실수로 계산한다.
평균은 소수점 둘째 자리까지 출력한다.

## 문제 2. 배열에서 최댓값 찾기

## 문제 설명
정수 10개를 입력받아 배열에 저장한 뒤,
배열에 들어 있는 값 중 가장 큰 값을 출력하는 프로그램을 작성하시오.

## 조건
정수는 총 10개 입력받는다.
입력받은 값은 배열에 저장한다.
반복문을 사용하여 최댓값을 찾는다.
배열의 첫 번째 값을 기준값으로 사용해도 된다.

## 문제 4. 배열 거꾸로 출력하기

## 문제 설명

정수 10개를 입력받아 배열에 저장한 뒤,
입력된 순서와 반대로 배열의 값을 출력하는 프로그램을 작성하시오.

## 조건
정수는 총 10개 입력받는다.
입력받은 값은 배열에 저장한다.
배열의 마지막 요소부터 첫 번째 요소까지 출력한다.
반복문을 사용한다.
